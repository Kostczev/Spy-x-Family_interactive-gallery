# Interactive Gallery Landing

Landing page with custom-built autoplay system and interactive gallery components.
Built without frameworks to demonstrate DOM architecture and state-driven UI logic.

## Stack
- HTML
- CSS
- Vanilla JavaScript

## Features
- Custom Autoplay Manager
- Hover pause / resume
- Page Visibility API handling
- Infinite carousel logic
- Three-item switcher with layered transitions
- Modular JS structure (group-based component management)
- No external libraries

## Architecture Notes
The autoplay logic is separated from UI bindings.
Components are registered in grouped managers to control lifecycle and behavior consistently.
